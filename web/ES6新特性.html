    # 4. Destructuring Assignment (解构赋值)

    一个简单的赋值例子，其中house和mouse是key，同时house和mouse也是一个变量，在ES5中是这样的：

    var data = $('body').data();	//data 拥有两个属性house 和 mouse
house = data.house;
mouse = data.mouse;

    在Node.js中用ES5是这样的：

    var jsonMiddleware = require('body-parser').jsonMiddleware;
var body = req.body;
username = body.username;
password = body.password;

    在ES6中，可以使用以下语句来代替上面的ES5代码：

    var { house,mouse } = $('body').data();
var {jsonMiddleware} = require('body-parser');
var {username,password} = req.body;

    这个同样也适用于数组：

    var [col1,col2] = $('.column'),
    [line1,line2,line3,,line5] = file.split('n');

    # 5. Enhanced Object Literals(增强的对象文本)

    # 1. 对象字面量扩展

    给一个属性赋一个变量值，如果变量名和属性名相同，可以省略变量名和冒号：

    let x = 1, y = 2;
let object = {
    x,
    y
};
console.log(object.x)  //'1'

    把一个匿名函数赋值给属性的时候可以简写：

    let object = {
    myFunction(){
        console.log('hello,world')
    }
}

    在ES6中，对象字面量中可以存在动态生成的属性，需要把动态生成的属性用[]包起来。

    # 2. 新的方法

    1. Object.is()方法，判断两个数是否相等。
    console.log(+0 === -0);	//true
console.log(Object.is(+0,-0));	//false
console.log(Object.is(NaN,NaN));	//false
    1. Object.assign()方法，用于将所有可枚举属性的值从一个或多个源对象复制到目标对象。他将返回目标对象。
    const target = {a:1,b:2};
const source = {b:4,c:5};
const returnedTarget = Object.assign(target,source);
console.log(target);	//Object{a:1,b:4,c:5}
console.log(returnedTarget);	//Object{a:1,b:4,c:5}
    1. Object.setPrototypeOf(obj,prototypeObj),设置一个对象的原型对象，他接受两个参数，一个是对象，一个是对象要链接到的原型对象。
    let person = {
    greeting(){
        return 'hello'
    }
}
let dog ={
    greeting(){
        return 'woof'
    }
}

let obj = {}

Object.setPrototypeOf(obj, person);
console.log(obj.greeting()); // "hello"
// obj 链接到dog
Object.setPrototypeOf(obj, dog);
console.log(obj.greeting()); // woof

    # 6. Arrow Functions(箭头函数)

    箭头函数的好处就是可以使this即使身处函数里面也能不改变指向。

    $('btn').click((event) =>{
    this.sendData();
})

    通过call传递文本给logUpperCase()函数：

    var logUpperCase = function(){
    this.string = this.string.toUpperCase();
    return () => console.log(this.string);
}
logUpperCase.call({string: 'ES 6 rocks'})();
