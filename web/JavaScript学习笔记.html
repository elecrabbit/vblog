# js介绍

js分为三个部分：

  1. ECMAScript标准——规定js的基本语法

var arr = [10,200,20,0,60,30];
	for(var i=0;i<arr.length-1;i++){
		for(var j = 0;j<arr.length-1-i;j++){
			if(arr[j]<arr[j+1]){		//从小至大排列用>

# 函数

  • 如果一个函数里有return，这个函数就有返回值
  • 如果一个函数里没有return，这个函数就没有返回值
  • 形参的个数可以不等于实参的个数
  • return后面的代码不会执行
  • 函数即可以作参数也可以作返回值使用

# 作用域

作用域即是变量的使用范围

  • 全局作用域：全局变量在任何位置都可以使用的范围
  • 局部作用域：局部变量只能在某个地方使用，比如函数内部
  • 作用域链：在一个函数中使用一个变量，先在这个函数中搜索这个变量，找到则使用，找不到则继续向外找这个变量，一直到全局作用域，还是找不到则是undefined。

# 预解析

  • 预解析把变量声明提前到了当前的作用域的最上面
  • 函数的声明也被提前到了当前作用域的最上面
  • 预解析会分段，多对script标签中的函数重名，预解析的时候不会冲突
  • 预解析只会提升声明不会提升赋值和调用
  • 先提升var，再提升function

console.log(obj.name);//获取属性值

//第二种方式：

一般首字母大写的函数是构造函数，是用来创建对象的。

创建对象的时候，系统做了以下4件事：

  1. 在内存中开辟空间，存储创建的新的对象

  3. 设置对象的方法和属性的值

# JSON数据

json也是一个对象，数据都是成对的，一般json格式的数据无论是键还是值都是用双引号括起来的。

遍历对象是不能for循环遍历的，可以通过for-in循环遍历。

var obj = {
    "name":"小明",
    "age":"18",
    "sex":"男"
};
for(var key in obj){
    console.log(key+"---"+obj[key]);//obj对象中的属性的名字
    //此时访问值不能用obj.key,只能用obj[key]。因为obj.key表示的是obj对象里的key属性，而此时没有key属性，所以obj.key=undefined;
}
/*结果：
name---小明
age---18
sex---男
*/

# String对象

String是一个对象。字符串可以看作是数组，所以可以通过for循环遍历，但是，字符串的值具有不可变性。

字符串可以通过索引访问字符串中的某个值，但是不能改变。

字符串的常用属性：

  • .Length——字符串的长度
  • .charAt(索引)——返回值是索引位置的字符串，超出索引返回的是空字符串
  • .fromCharCode(数字值，可以是多个值)——返回的是ASCII码对应的字符
  • .concat(字符串1，字符串2，……)——返回拼接好的字符串
  • .indexOf(要找的字符串，[从某处开始]) ——返回的是这个字符串的索引值，没有则返回-1
  • .lastIndexOf(要找的字符串，[开始的索引值]) ——同.indexOf一样，只是从后向前查找
  • .match()——检索字符串与正则表达式匹配的匹配项
  • .repalce(原来的字符串，新的字符串)——用来替换字符串的，原来的字符串不会改变
  • .slice(开始的索引，结束的索引)——提取两个索引之间的字符串
  • .split(字符串，返回的字符串长度)——用指定的分隔符分割字符串
  • .substr(开始的位置，个数)——返回截取后的新的字符串
  • .substring(开始的索引，结束的索引)——返回截取后的字符串，不包含结束索引的字符串
  • .toLowerCase()——转小写
  • .toUpperCase()——转大写
  • .trim()——去掉字符串两端的空格

# Array对象

  • Array.isArray(对象)——判断这个对象是不是数组
  • .concat(数组，数组……)——数组拼接
  • .every(函数)——返回布尔值，函数中有三个参数，第一个是元素的值，第二个是索引值，第三个是原来的数组
  • .filter(函数)——返回每个符合条件的元素组成一个新的数组
  • .push(值)——把值追加到数组最后位置，并返回当前数组的长度
  • .pop()——删除最后一个元素，并返回删除的这个值
  • .shift()——删除第一个元素，并返回删除的这个值
  • .unshift(值)——在数组的第一个元素前面插入值，并返回当前的数组长度
  • .forEach(函数)——遍历数组，相当于for循环
  • .indexOf(元素值)——返回索引，没有则返回-1
  • .join("字符串")——返回一个拼接的字符串
  • .map()——创建一个新的数组，返回数组中每个元素调用函数的结果

# 事件的绑定

# 为同一个事件绑定多个相同事件

三种方式：

  1. 对象.on事件名字 = 事件处理函数，多个相同事件注册时，执行的是最后一个，之前的被覆盖了：

    $("btn").onclick = function(){};

  2. 对象.addEventListener("没有on的事件名字"，事件处理函数，false);

    $("btn").addEventListener("click",function(){},false);

  3. 对象.attachEvent("没有on的事件名字"，事件处理函数);

    $("btn").attachEvent("onclick",funtion(){});

# 事件的解绑

用什么方式绑定事件，就应该用对应的方式解绑事件：

  1. 对象.on事件名字 = 事件处理函数-->绑定事件

    对象.on事件名字 = null;-->解绑事件

  2. 对象.addEventListener("没有on的事件类型",命名函数,false);-->绑定事件

    对象.removeEventListener("没有on的事件类型",命名函数,false);-->解绑事件

  3. 对象.attachEvent("on事件类型",命名函数);--->绑定事件

    对象.detachEvent("on事件类型",函数名字);--->解绑事件

# 定时器

# setTimeout()和clearTimeout()

在指定的毫秒数到达之后执行指定的函数，只执行一次

<script>
    //创建一个定时器，1s后执行，返回定时器的标示
    var timerId = setTimeout(function(){
        console.log("Hello World");
    },1000);
    //取消定时器的执行
    clearTimeout(timerId);
</script>

//创建一个定时器，每隔1s调用一次
    var timerId = setInterval(function(){
        var date = new Date();
        console.log(date.tolocalTimeString());
    },1000);
    //取消定时器的执行
    clearInterval(timerId);
</script>

# URL

URL的组成：

scheme://host:port/path?query#fragment

scheme通信协议，常用的有http、ftp等

host主机，服务器的ip地址

port端口号，http的默认端口号为80

path路径，由0个或多个“/”隔开的字符串，一般表示主机的目录或文件地址

query查询，用于给动态网页传递参数，用“&”隔开，参数名和数值用“=”隔开。

fragment信息片断，通常为字符串、锚点等
